# Automotive-TSN-Switch
TSN Switch for Automotive, Industry. gPTP Time Synchronization for autonomous driving. such as RoboVAN, RoboTruck, RoboBus, RoboTaxi, Robots, eVTOL, etc.

SW2000M TSN:
11 port Gigabit/100Mbps Automotive Ethernet TSN time sensitive switch, which includes 8 channels of 100/1000BASE-T1 using Tyco MATEnet interface, 1 channel 1/2.5/5/10G standard Ethernet SFP+ and 2 channels 1000BASE-T standard Ethernet RJ45, it can achieve multi-channel switching of Automotive Ethernet, Gateway, AVB application, gPTP boundary clock (BC), gPTP clock bridge.
https://github.com/hdn-vdo/SW2000M-TSN-Switch

SW2000H TSN:
11 port Gigabit/100Mbps Automotive Ethernet TSN time sensitive switch, which includes 8 channels of 100/1000BASE-T1 using Rosenborg H-MTD, 1 channel 1/2.5/5/10G standard Ethernet SFP+ and 2 channels 1000BASE-T standard Ethernet RJ45, it can achieve multi-channel switching of Automotive Ethernet, Gateway, AVB application, gPTP boundary clock (BC), gPTP clock bridge.

SW2000R TSN:
11 port Gigabit/100Mbps Automotive Ethernet TSN time sensitive switch, which includes 8 channels of 100/1000BASE-T RJ45,2 channel 1/2.5/5/10G standard Ethernet SFP+, 1 channel 1/2.5G standard Ethernet SFP+, it can achieve multi-channel switching of Ethernet Gateway, AVB application, gPTP boundary clock (BC), gPTP clock bridge.

SW2000G-M12X TSN:
11 port Gigabit/100Mbps Automotive Ethernet TSN time sensitive switch, which includes 8 channels of 100/1000BASE-T M12X-Code, it can achieve multi-channel switching of Ethernet Gateway, AVB application, gPTP boundary clock (BC), gPTP clock bridge.

SW1100TSN:
7-port 100base-t1 automotive Ethernet switch, 5 channels of 100BASE-T1 and 1 channel of 1000BASE-T1 TE MATEnet, with 1 channel 100/1000BASE-T standard Ethernet (RJ45 interface). It enables nanosecond-level time synchronization, multi-channel switching for automotive Ethernet, bypass data acquisition, and monitoring functions. The switch supports TSN/AVB protocols, boundary clocks, and gPTP clock bridging, complying with the IEEE 802.1AS-2020 standard. Its clock Domain ID ranges from 0 to 255. Additionally, it fully supports the AVB protocol suite, enabling comprehensive audio/video bridging capabilities.

SW3000R TSN:
The 30-port high-performance TSN (Time-Sensitive Networking) switch is equipped with 25 channels of Gigabit adaptive RJ45 interfaces (with optional PoE support), 4 channels of 1/2.5/5/10G multi-speed SFP+ interfaces, and one PCIe-endpoint interface. It supports SyncE frequency synchronization, IEEE 1588-PTP, and 802.1AS-gPTP. It also provides 1PPS/ToD input/output and a 10MHz atomic clock output. Supports L2 switching and L3 routing, along with complex management features such as 802.1Q, QoS, and port control. Suitable for automotive/aeronautical gateway and AVB implementations. Additionally, it supports gPTP Boundary Clock (BC) functionality, a full suite of TSN protocols, and HSR/PRP (RedBox) link redundancy mechanisms.
